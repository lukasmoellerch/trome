# trome

TUI Chrome. Browse the web in your terminal.

Uses Playwright to render pages headlessly and displays them using half-block characters at 30fps.

## Usage

### Quick Start (bunx)

```bash
bunx github:lukasmoellerch/trome https://example.com
```

### From Source

```bash
bun install
bun run dev https://example.com
```

## Terminal Support

Works well in [Ghostty](https://ghostty.org/). Other terminals are unsupported.

## Controls

- Scroll with mouse wheel
- Click with mouse
- Type with keyboard
- Escape or Ctrl+C to quit
